NIH-Chest-X-Ray-MultiClass-Classification

This project builds a deep learning model to classify chest X-ray images into 14 disease categories using DenseNet-121. Trained on the NIH Chest X-ray dataset, the model employs a resampling technique to handle class imbalance. Performance is evaluated using AUC and ROC AUC scores for accurate multi-class classification.

Technologies/Methods Used: Python, TensorFlow, DenseNet-121, CNN, Image Processing, Resampling
